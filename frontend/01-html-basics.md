# HTML Basics

## What is HTML?
HTML is the structure of a webpage - like the skeleton of a house.

## Key Concepts:

### 1. Basic Structure
Every HTML file needs:
- `<!DOCTYPE html>` - tells browser this is HTML5
- `<html>` - root element
- `<head>` - metadata (title, links to CSS)
- `<body>` - visible content

### 2. Common Elements
- `<div>` - container/box
- `<h1>`, `<h2>`, etc. - headings
- `<p>` - paragraph
- `<button>` - clickable button
- `<input>` - text input field
- `<ul>`, `<li>` - lists
- `<span>` - inline container

### 3. Attributes
- `id="something"` - unique identifier
- `class="something"` - reusable identifier
- `onclick="functionName()"` - runs function when clicked

### Example:tml
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <button id="myButton">Click Me</button>
</body>
</html>
