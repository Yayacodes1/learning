## Step-by-Step:

### Phase 1: Database (Day 1-2)
1. Sign up for Supabase
2. Create database
3. Create users and tasks tables
4. Get connection string

### Phase 2: Backend (Day 3-5)
1. Set up Express server
2. Connect to PostgreSQL
3. Create registration endpoint
4. Create login endpoint
5. Add authentication middleware
6. Create protected task routes

### Phase 3: Frontend (Day 6-8)
1. Create login page
2. Create main app page
3. Connect to backend API
4. Add authentication flow
5. Test everything

### Phase 4: Deploy (Day 9-10)
1. Deploy backend to Railway
2. Deploy frontend to Netlify
3. Update API URLs
4. Test live app
